# Ürün Yönetim Motoru
1. **Dinamik Kategori:** Kategorileri benzersizleştir ve alfabetik olarak sırala.
2. **Resim İşleme:** `FileReader` ile base64 dönüşümü yap.
3. **Senkronizasyon:** `localStorage.setItem` ile JSON verisini güncelle.
